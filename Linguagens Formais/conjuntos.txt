AULA 2



Uma relação binária de A para B é um subconjunto de AxB

A={a,b} B={c,d} AxB={(a,c),(a,d),(b,c),(b,d)} e a relação R, pode ser por exemplo = {(a,d),{b,d)}]

dominio de R é o conjunto = {a,b} --> que são os x

contradominio de R é o conjunto = {d} --> que são os y



relação inversa de R = R^(-1) = {(y,x)|(x,y) pertencente a R}

relação sobre A --> se A=B entãa a relação é denominada relação sobre A

relação identidade --> Ia={(x,x)|x pertence a A} --> {(a,a),(b,b)}

numero maximo de relações binarias sobre A é 2^n^2



representação por matriz -->   c  d  --> as linhas representam os dominios

			     a 0  1  --> as colunas representam os contradominios

			     b 0  1

representação por dígrafo -->   a-->d   c

				b---^



relação reflexiva --> tem todas os subconjuntos (x,x), onde x vai para ele mesmo

relação antirreflexiva --> se nem todos os subconjuntos são (x,x), onde x vai para ele mesmo

relação simétrica --> se todos os casos de (x,y) tem um (y,x), onde x vai de volta de y

relação antissimétrica --> se nem todos os casos de (x,y) tem um (y,x), onde x vai e volta de y

relação transitiva --> se a mais de 1 forma de chegar de um ponto a outro(2 caminhos distintos)



composição de relações --> R o S --> R^0 = Ia, R^1 = R, R^2 = R o R, R^3 = R o R o R ...

	EX: R={(1,1),(1,2),(2,3),(3,3)}, R^2={(1,2),(1,3),(2,3)}, R^3={(1,3)}, R^4 = conjunto vazio

fecho transitivo --> R^+ = a união de todos os Rs^n começando em n=1

fecho transitivo reflexivo --> R^* = a união de todos os Rs^n começando em n=0

no feicho transitivo você consegue definir quantas viagens é necessario fazer para chegar de um x a

	um certo y, logo, se em R^1 tiver um subconjunto (x,y) significa que em apenas uma "viagem"

	é possivel chegar em y a partir de x, então se em R^3 tiver um subconjunto (x,y) significa

	que em 3 "viagens" é possivel chegar em y a partir de x.
